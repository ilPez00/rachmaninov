# ◈ The Rachmaninov Ecosystem: Sovereign Intelligence

**Note:** This SD card contains a standalone core sample of a much larger federated ecosystem.

## 🌌 The Broader Project
This system is designed as a "Cybernetic Framework for the Will," consisting of two primary interfaces powered by the core logic found on this card:

1.  **Aura (Field Intelligence):** An Android-based Augmented Reality (AR) HUD. It treats the smartphone as a sensor array (Vision, Audio, Biometrics) to provide real-time context-aware coaching.
2.  **Praxis (Accountability Hub):** A federated web platform for social goal-setting and "skin in the game" accountability bets.

## 🎹 Core Components (Included on this Card)

### 1. Rachmaninov (The Ontological Engine)
Rachmaninov is the deterministic heart of the system. It maps chaotic human language to a structured **Action Ontology**. It bridges high-level intent with low-level execution, ensuring every action is graded and checked within a **PDCA (Plan-Do-Check-Act)** loop.

### 2. Lattice (The Federated Bridge)
Lattice is the protocol that allows Aura and Praxis to communicate. It enables **anonymized data federation**, where local sensor logs from Aura are synthesized into abstract patterns and synced to Praxis without ever exposing the user's private data or raw text.

### 3. Dreaming (Adaptive Variation)
When the system detects a "stalled" goal (via the PDCA loop), it enters a **Dreaming State**. Using the local knowledge base, it generates virtual variations of the goal — "hypothetical pivots" — to find a new path forward when traditional methods fail.

### 4. UberWiki (Semantic Knowledge Base)
A two-layer compressed wiki optimized for small, local AI models.
- **Human Layer:** Human-readable Markdown files.
- **AI Layer:** A vectorized, hierarchical trie that allows for constant-cost context retrieval, solving the long-term context window problem.
